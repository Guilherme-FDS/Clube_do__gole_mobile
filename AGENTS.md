# Versão do Expo travada em SDK 54

Este projeto está pinado no **Expo SDK 54** (`~54.0.35`) de propósito — é a versão que o app **Expo Go** da App Store suporta atualmente. Não faça upgrade de `expo`, `react-native`, `react` ou dos pacotes `expo-*`/`react-native-*` pra versões de SDKs mais novos (55, 56...) sem confirmar antes — isso quebra o Expo Go do iPhone com erro de conflito de versão.

Antes de escrever código, consulte os docs versionados da SDK 54: https://docs.expo.dev/versions/v54.0.0/
